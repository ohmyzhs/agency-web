# Posts 카테고리 개편과 콘텐츠 자동화 준비

Zero Human Studio의 글 아카이브를 더 읽기 쉬운 구조로 정리했습니다. 기존의 내부 작업 분류를 그대로 노출하기보다, 방문자가 글의 목적을 바로 이해할 수 있도록 public category를 단순화했습니다.

## 바뀐 점

- Posts의 대표 카테고리를 AI 인사이트, 실용 가이드, 비교·추천, 업무 생산성, 디지털 트렌드로 정리했습니다.
- 제작 기록과 실험 글은 메인 글 목록에서는 디지털 트렌드와 실용적인 맥락 안에서 읽히도록 조정했습니다.
- 사이트 기능 추가와 운영 변경사항은 별도 Updates 영역에서 짧게 확인할 수 있도록 준비했습니다.

## 앞으로의 방향

이 구조는 외부 AI 도구나 자동화 워크플로우가 글 초안을 제출하더라도, 같은 카테고리와 품질 기준으로 검수할 수 있게 만드는 기반입니다. 다음 단계에서는 Git 기반 콘텐츠 제출 API를 통해 Markdown 글과 업데이트 노트를 안전하게 받을 수 있도록 확장할 예정입니다.
